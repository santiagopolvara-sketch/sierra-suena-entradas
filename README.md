# Sierra Suena Entradas 🎵

La Primavera vuelve a sonar en Sierra de la Ventana con dos días a pura música.

Sierra Suena larga su primera edición los días **sábado 19 y domingo 20 de septiembre de 2026**. Una fecha que se caracterizará por espectáculos de alto calibre y la recuperación de una tradición histórica.

## 🎪 Sobre el evento

Sierra Suena te invita a un fin de semana para toda la familia, con una grilla artística de primer nivel y un predio ubicado en un entorno natural inigualable.

Preparate solo para disfrutar:
- 🎶 Buena música
- 🍽️ Patio gastronómico para todos los gustos
- 🌄 Entorno natural inigualable
- ✨ Y mucho más!

## 🛠️ Stack Tecnológico

### Frontend
- **React 18** + TypeScript
- **Tailwind CSS** para estilos
- **Google Sans Flex** tipografía
- Colores personalizados según manual de marca

### Backend
- **Node.js** + Express
- **PostgreSQL** base de datos
- **JWT** + **OAuth Google** autenticación
- **MercadoPago** integración de pagos

## 📋 Estructura del Proyecto

```
sierra-suena-entradas/
├── frontend/          # React app
├── backend/           # Node.js Express API
├── docs/              # Documentación
└── README.md
```

## 🎟️ Precios de Entradas

- **Junio**: Valor por 2 días
- **Julio - Agosto**: Valor por día individual O combo 2 días
- **Septiembre - Octubre**: Valor diferente (evento principal)
